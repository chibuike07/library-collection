The Developer

My name is Chime Chibuike Princewill, am an indigen of Umudim Amokwe in Udi LGA Enugu, am a web delevoper and a site manager, i code with HTML , CSS, JAVASCRIPT. Presently i am a serving corper under Obafemi Owode Local Government and currently serving under the jurisdiction of Mowe , in CHRISTOPHER UNIVERSITY precisely

Here is an application base work build for to put and get library lists of collections. Inside it holds pages of the school, a brief explaination of what library is all about,it's departments and the roles they play, it also has the list of collections based on the classes which the viewer can filter through to get the class of collection to which users search can be traced and further filter to the authors of a book, it carries add list box that enable the library officials to add new entries of their collection

I started this project because since i was posted to this institution to serve my beloved country by the National Youth Corp Service, which i served in the university library known as Nelson Mandela Library , i saw and notice that there was really much to do in the library considering the load of work and less man power and always feeling how my boss works hard each day she comes around and how she really wish that things will be done through the use of an application that will allow her to access and retrieve library materials easily and to be able to meet up with the patrons search / request of information, but there was no much finance availabe to enable a developer to build an application that will be satisfied , flexible and suitable for the library uses. I saw the general problem and saw it as a common goal that needs all hands on deck, so i took it upon my self to help to achieve the said common goal, i started puting my hard work day and night to come up with this beautiful application that will assisst and satisfy the accessibility of library holdings.

In conclusion this application is meant for the library officials uses only
